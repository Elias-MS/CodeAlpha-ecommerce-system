## ✨ Animation Effects

### 1. **fadeIn** (Page Load)
```
From: opacity 0, translateY(20px)
To: opacity 1, translateY(0)
Duration: 0.5s
```

### 2. **slideInUp** (Content)
```
From: opacity 0, translateY(40px)
To: opacity 1, translateY(0)
Duration: 0.6s
```

### 3. **shimmer** (Card Headers)
```
Light sweep effect from left to right
Duration: 3s infinite
```

### 4. **Hover Effects**
- Cards: translateY(-10px) + scale(1.02)
- Buttons: translateY(-5px) + scale(1.05)
- Images: scale(1.05) + larger shadow
- Links: Animated underline
