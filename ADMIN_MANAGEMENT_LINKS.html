📖 How to Use Admin Features

🔧 Product Management:

  1. Go to All Products
  2. To activate: Select products → Actions → "Activate selected products"
  3. To deactivate: Select products → Actions → "Deactivate selected products"
  4. To edit: Click on product name
  5. To delete: Select products → Actions → "Delete selected products"
  6. To add: Click "Add Product" button

👥 User Management:

  1. Go to All Users
  2. To activate: Select users → Actions → "Activate selected users"
  3. To deactivate: Select users → Actions → "Deactivate selected users"
  4. To make staff: Select users → Actions → "Make staff"
  5. To remove staff: Select users → Actions → "Remove staff status"
  6. To edit: Click on username

💬 Complaint Management:

  1. Go to All Complaints
  2. Click on a complaint to view details
  3. Write your reply in "Admin reply" field
  4. Change status to "Replied" or "Resolved"
  5. Check "is_urgent" to create public announcement
  6. Click Save
