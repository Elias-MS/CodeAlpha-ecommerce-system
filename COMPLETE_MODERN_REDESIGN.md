## 🎨 Design System

### Color Palette
```css
Primary Gradient:  #667eea → #764ba2 (Purple)
Secondary:         #ff6b35 (Orange - for prices)
Accent:            #ffeaa7 (Yellow - for ratings)
Success:           #00b894 (Green)
Text Dark:         #2d3436
Text Light:        #636e72
Background:        #f8f9fa
White:             #ffffff
```

### Typography
```css
Hero Title:        3.5rem, weight 800
Section Title:     2.5rem, weight 700
Product Name:      2.2rem, weight 700
Large Price:       3rem, weight 800
Regular Price:     1.5rem, weight 700
Body Text:         1.05rem, line-height 1.7
```

### Border Radius
```css
Cards:             16px - 20px
Buttons:           10px - 50px (rounded)
Badges:            12px - 20px
Input Fields:      10px - 50px
```

### Shadows
```css
Light:             0 2px 8px rgba(0,0,0,0.06)
Medium:            0 4px 12px rgba(0,0,0,0.08)
Heavy:             0 8px 24px rgba(0,0,0,0.12)
Hover:             0 12px 28px rgba(0,0,0,0.15)
```

### Animations
```css
Transition:        all 0.3s ease
Hover Lift:        translateY(-8px)
Image Zoom:        scale(1.08)
Button Lift:       translateY(-3px)
Float Animation:   3s ease-in-out infinite
```
