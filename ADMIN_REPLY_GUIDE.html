💬 Admin Reply Guide

How to Reply Privately to User Complaints

🚀 Quick Start

📍 Admin Panel URL: http://127.0.0.1:8000/admin/
📍 Complaints Section: http://127.0.0.1:8000/admin/users/complaint/
🔗 Open Complaints in Admin Panel

📋 Step-by-Step Guide

1 Login to Admin Panel

Go to: http://127.0.0.1:8000/admin/

Login with your admin credentials:

  • Username: admin
  • Password: admin123
2 Navigate to Complaints

In the admin panel, find the "USERS" section

Click on "Complaints"

You'll see a list of all user complaints

3 Select a Complaint

Click on any complaint to view full details

You'll see:

  • User who submitted the complaint
  • Subject and detailed message
  • Uploaded image (if any)
  • Current status
  • Submission date
4 Write Your Private Reply

Scroll down to the "Admin Response" section

In the "Admin reply" textarea, type your private message to the user

Example Admin Reply
Admin reply:
Dear [User Name],

Thank you for bringing this to our attention. We have reviewed your complaint and found that [explanation].

We have taken the following actions:
- [Action 1]
- [Action 2]

[Resolution details]

If you have any further questions, please don't hesitate to contact us.

Best regards,
Customer Support Team
5 Update Status

Change the "Status" dropdown:

  • Pending - Not yet reviewed
  • Replied - Admin has replied (auto-set when you add reply)
  • Resolved - Issue is completely resolved
6 Mark as Urgent (Optional)

If this complaint affects multiple users and needs public announcement:

  • Check the "Is urgent" checkbox
  • This will create a public announcement automatically
  • All users will see it at /users/announcements/
⚠️ Note: Only mark as urgent if the issue affects multiple users (e.g., payment gateway down, shipping delays, etc.)
7 Save the Complaint

Click the "Save" button at the bottom

The system will automatically:

  • Set the "Replied at" timestamp
  • Change status to "Replied" if it was "Pending"
  • Create public announcement if marked as urgent
8 User Sees Your Reply

The user will now see:

  • ✓ Admin replied privately badge in their complaints list
  • Your full reply when they click "View Details & Reply"
  • Reply timestamp showing when you responded

🔒 Privacy & Security

🔐 Private Replies

Your replies are PRIVATE and only visible to:

  • ✅ The user who submitted the complaint
  • ✅ Admin users in the admin panel

Your replies are NOT visible to:

  • ❌ Other regular users
  • ❌ Public visitors
  • ❌ Search engines
📢 Public Announcements

Only created when you check "Is urgent"

  • Used for issues affecting multiple users
  • Visible to all users at /users/announcements/
  • Example: "Payment gateway maintenance scheduled"

✨ Best Practices

1. Be Professional & Courteous

  • Address the user by name if possible
  • Thank them for bringing the issue to your attention
  • Use polite and professional language
  • Sign off with your name or team name

2. Be Clear & Specific

  • Explain what the issue was
  • Detail what actions you've taken
  • Provide reference numbers if applicable
  • Give timelines for resolution

3. Be Helpful

  • Offer solutions or alternatives
  • Provide contact information for further help
  • Include relevant links or resources
  • Ask if they need additional assistance

4. Response Time

  • Try to respond within 24-48 hours
  • For urgent issues, respond within a few hours
  • If investigation takes time, send an acknowledgment first

Quality Assurance Team
