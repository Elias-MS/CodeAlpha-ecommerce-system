┌────────────────────────────────────────────────────────────────┐
│  📖 HOW TO USE - PRODUCT MANAGEMENT                            │
└────────────────────────────────────────────────────────────────┘

   1. Go to: http://127.0.0.1:8000/admin/products/product/
   
   2. TO ACTIVATE PRODUCTS:
      • Select products (checkboxes)
      • Choose "Activate selected products" from Actions
      • Click "Go"
      • Products will have stock set to 10
   
   3. TO DEACTIVATE PRODUCTS:
      • Select products (checkboxes)
      • Choose "Deactivate selected products" from Actions
      • Click "Go"
      • Products will have stock set to 0 (out of stock)
   
   4. TO EDIT A PRODUCT:
      • Click on product name
      • Update fields
      • Click "Save"
   
   5. TO DELETE PRODUCTS:
      • Select products (checkboxes)
      • Choose "Delete selected products" from Actions
      • Confirm deletion
   
   6. TO ADD NEW PRODUCT:
      • Click "Add Product" button
      • Fill in all fields
      • Upload image
      • Click "Save"


┌────────────────────────────────────────────────────────────────┐
│  📖 HOW TO USE - USER MANAGEMENT                               │
└────────────────────────────────────────────────────────────────┘

   1. Go to: http://127.0.0.1:8000/admin/auth/user/
   
   2. TO ACTIVATE USERS:
      • Select users (checkboxes)
      • Choose "Activate selected users" from Actions
      • Click "Go"
   
   3. TO DEACTIVATE USERS:
      • Select users (checkboxes)
      • Choose "Deactivate selected users" from Actions
      • Click "Go"
      • Note: Cannot deactivate superusers
   
   4. TO MAKE STAFF:
      • Select users (checkboxes)
      • Choose "Make staff" from Actions
      • Click "Go"
   
   5. TO REMOVE STAFF:
      • Select users (checkboxes)
      • Choose "Remove staff status" from Actions
      • Click "Go"
      • Note: Cannot remove staff from superusers


┌────────────────────────────────────────────────────────────────┐
│  📖 HOW TO USE - COMPLAINT MANAGEMENT                          │
└────────────────────────────────────────────────────────────────┘

   1. Go to: http://127.0.0.1:8000/admin/users/complaint/
   
   2. TO REPLY TO COMPLAINT:
      • Click on complaint
      • Write reply in "Admin reply" field
      • Change status to "Replied"
      • Click "Save"
   
   3. TO CREATE PUBLIC ANNOUNCEMENT:
      • Click on complaint
      • Write reply in "Admin reply" field
      • Check "is_urgent" checkbox
      • Click "Save"
      • Announcement will be created automatically
